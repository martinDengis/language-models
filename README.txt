# Language Models Project

This repository contains the code and resources for the Language Models project, part of the Master 3 program.

## Project Structure

- `data/`: Contains datasets used for training and evaluation.
- `models/`: Contains the implementation of various language models.
- `notebooks/`: Jupyter notebooks for experiments and analysis.
- `scripts/`: Utility scripts for data preprocessing, training, and evaluation.
- `results/`: Directory to store the results of experiments.
- `README.txt`: This file, providing an overview of the repository.

## Getting Started

### Prerequisites

- Python 3.8 or higher
- Required Python packages (listed in `requirements.txt`)

### Installation

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/language-models.git
    ```
2. Navigate to the project directory:
    ```
    cd language-models
    ```
3. Install the required packages:
    ```
    pip install -r requirements.txt
    ```
